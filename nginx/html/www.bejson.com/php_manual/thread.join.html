Thread::join

(PECL pthreads >= 2.0.0)

Thread::join — 同步

说明

publicThread::join(): bool

让当前执行上下文等待被引用线程执行完毕

参数

此函数没有参数。

返回值

成功时返回 true， 或者在失败时返回 false。

范例

示例 #1 加入线程

<?php
class My extends Thread {
    public function run() {
        /* ... */
    }
}
$my = new My();
$my->start();
/* ... */
var_dump($my->join());
/* ... */
?>

以上例程会输出：

bool(true)
