curl_share_strerror

(PHP 7 >= 7.1.0, PHP 8)

curl_share_strerror — 返回错误号对应的错误消息

说明

curl_share_strerror(int $errornum): string

返回错误号对应的错误消息。

参数

errornum

» cURL error codes 常量。

返回值

返回错误消息。如果传入了无效的错误号，返回 null。
