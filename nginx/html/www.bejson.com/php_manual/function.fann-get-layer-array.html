fann_get_layer_array

(PECL fann >= 1.0.0)

fann_get_layer_array — 获取网络中每层的神经元数量。

说明

fann_get_layer_array(resource $ann): array

获取网络中每层的神经元数量。

偏差神经元不包括在内，所以层数是和 fann_create 函数组想匹配的。

参数

ann

神经网络 资源。

返回值

包含每层神经元数量的数组。
