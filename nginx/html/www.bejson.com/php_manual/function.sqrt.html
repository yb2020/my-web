sqrt

(PHP 4, PHP 5, PHP 7, PHP 8)

sqrt — 平方根

说明

sqrt(float $arg): float

返回 arg 的平方根。

参数

arg

要处理的参数

返回值

返回 arg 的平方根，负数时返回 NAN。

范例

示例 #1 sqrt() 例子

<?php
// Precision depends on your precision directive
echo sqrt(9); // 3
echo sqrt(10); // 3.16227766 ...
?>
