iconv_set_encoding

(PHP 4 >= 4.0.5, PHP 5, PHP 7, PHP 8)

iconv_set_encoding — 为字符编码转换设定当前设置

说明

iconv_set_encoding(string $type, string $charset): bool

将 type 设置的值从内部配置变量更改为 charset。

参数

type

type 的值可以是以下其中任意一个：

  • input_encoding
  • output_encoding
  • internal_encoding

charset

字符集。

返回值

成功时返回 true， 或者在失败时返回 false。

范例

示例 #1 iconv_set_encoding() 例子

<?php
iconv_set_encoding("internal_encoding", "UTF-8");
iconv_set_encoding("output_encoding", "ISO-8859-1");
?>

参见

  • iconv_get_encoding() - 获取 iconv 扩展的内部配置变量
  • ob_iconv_handler() - 以输出缓冲处理程序转换字符编码
