header_remove

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

header_remove — 删除之前设置的 HTTP 头

说明

header_remove(string $name = ?): void

删除之前用 header() 设置的 HTTP 头。

参数

name

要移除的头名称。

注意: 参数不分大小写。

返回值

没有返回值。

范例

示例 #1 取消指定的头

<?php
header("X-Foo: Bar");
header("X-Bar: Baz");
header_remove("X-Foo"); 
?>

以上例程的输出类似于：

X-Bar: Baz

示例 #2 取消之前全部指定的头

<?php
header("X-Foo: Bar");
header("X-Bar: Baz");
header_remove(); 
?>

以上例程的输出类似于：


    

   

  

 



 

  
注释

  
警告
   

    本函数会删除所有 PHP 设置的头，
    包括 Cookie、Session 和 X-Powered-By。
   

  

  
注意: 


 数据头只会在SAPI支持时得到处理和输出。
