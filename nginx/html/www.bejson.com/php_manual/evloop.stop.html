EvLoop::stop

(PECL ev >= 0.2.0)

EvLoop::stop — Stops the event loop

说明

public EvLoop::stop( int $how = ?): void

Stops the event loop

参数

how

One of Ev::BREAK_* constants .

返回值

没有返回值。
