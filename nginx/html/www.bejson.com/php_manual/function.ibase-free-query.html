ibase_free_query

(PHP 5, PHP 7 < 7.4.0)

ibase_free_query — Free memory allocated by a prepared query

说明

ibase_free_query(resource $query): bool

Frees a prepared query.

参数

query

A query prepared with ibase_prepare().

返回值

成功时返回 true， 或者在失败时返回 false。
