Imagick::getImageScene

(PECL imagick 2, PECL imagick 3)

Imagick::getImageScene — Gets the image scene

说明

public Imagick::getImageScene(): int

Gets the image scene.

参数

此函数没有参数。

返回值

Returns the image scene.

错误／异常

错误时抛出 ImagickException。
