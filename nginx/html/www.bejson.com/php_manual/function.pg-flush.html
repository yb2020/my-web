pg_flush

(PHP 5 >= 5.6.0, PHP 7, PHP 8)

pg_flush — 刷新链接中已处理的数据查询

说明

pg_flush(resource $connection): mixed

pg_flush() 刷新任何已经处理的等待发送到链接的数据查询。

参数

connection

PostgreSQL 数据库链接资源

返回值

如果刷新成功或者没有数据等待刷新返回 true ， 如果返回 0 为部分刷新或者更多未被刷新,刷新失败为 false
