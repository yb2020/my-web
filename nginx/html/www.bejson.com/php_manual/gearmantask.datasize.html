GearmanTask::dataSize

(PECL gearman >= 0.5.0)

GearmanTask::dataSize — Get the size of returned data

说明

public GearmanTask::dataSize(): int

Returns the size of the data being returned for a task.

参数

此函数没有参数。

返回值

The data size, or false if there is no data.
