sinh

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

sinh — 双曲正弦

说明

sinh(float $arg): float

返回 arg 的双曲正弦值，定义为 (exp(arg) - exp(-arg))/2。

参数

arg

要处理的参数

返回值

arg 的双曲正弦值

参见

  • sin() - 正弦
  • asinh() - 反双曲正弦
  • cosh() - 双曲余弦
  • tanh() - 双曲正切
