is_infinite

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

is_infinite — 判断是否为无限值

说明

is_infinite(float $val): bool

如果 val 为无穷大（正的或负的），例如 log(0) 的结果或者任何超出本平台的浮点数范围的值，则返回 true。

参数

val

要检查的值

返回值

如果 val 为无穷大返回 true，否则返回 false。

参见

  • is_finite() - 判断是否为有限值
  • is_nan() - 判断是否为合法数值
