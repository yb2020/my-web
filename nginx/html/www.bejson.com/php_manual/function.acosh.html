acosh

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

acosh — 反双曲余弦

说明

acosh(float $arg): float

返回 arg 的反双曲余弦值，即，其双曲余弦为 arg 的那个值。

参数

arg

要处理的值

返回值

The inverse hyperbolic cosine of arg
