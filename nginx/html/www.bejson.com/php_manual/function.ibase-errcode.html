ibase_errcode

(PHP 5, PHP 7 < 7.4.0)

ibase_errcode — Return an error code

说明

ibase_errcode(): int

Returns the error code that resulted from the most recent InterBase function call.

参数

此函数没有参数。

返回值

Returns the error code as an integer, or false if no error occurred.
