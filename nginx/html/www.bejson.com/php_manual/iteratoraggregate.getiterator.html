IteratorAggregate::getIterator

(PHP 5, PHP 7, PHP 8)

IteratorAggregate::getIterator — 获取一个外部迭代器

说明

public IteratorAggregate::getIterator(): Traversable

返回一个外部迭代器。

参数

此函数没有参数。

返回值

实现了 Iterator 或 Traversable 接口的类的一个实例。

错误／异常

失败时抛出 Exception。
