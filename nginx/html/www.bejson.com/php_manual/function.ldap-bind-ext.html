ldap_bind_ext

(PHP 7 >= 7.3.0, PHP 8)

ldap_bind_ext — Bind to LDAP directory

说明

ldap_bind_ext(
    LDAP\Connection $ldap,
    ?string $dn = null,
    ?string $password = null,
    ?array $controls = null
): LDAP\Result|false

Does the same thing as ldap_bind() but returns an LDAP\Result instance to be parsed with ldap_parse_result().

参数

See ldap_bind()

返回值

Returns an LDAP\Result instance, 或者在失败时返回 false.

更新日志

版本 说明
8.1.0 现在 ldap 参数接受 LDAP\Connection 实例，之前接受 资源(resource)。
8.1.0 Returns an LDAP\Result instance now; previously, a 资源(resource) was returned.
8.0.0 controls is nullable now; previously, it defaulted to [].
