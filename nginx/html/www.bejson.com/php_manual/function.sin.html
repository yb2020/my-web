sin

(PHP 4, PHP 5, PHP 7, PHP 8)

sin — 正弦

说明

sin(float $arg): float

sin() 返回参数 arg 的正弦值。参数 arg 的单位为弧度。

参数

arg

单位是弧度的值。

返回值

arg 的正弦值

范例

示例 #1 sin() 例子

<?php

// 返回值的精度由配置中的 precision 指示确定
echo sin(deg2rad(60));  //  0.866025403 ...
echo sin(60);           // -0.304810621 ...

?>

参见

  • asin() - 反正弦
  • sinh() - 双曲正弦
  • cos() - 余弦
  • tan() - 正切
  • deg2rad() - 将角度转换为弧度
