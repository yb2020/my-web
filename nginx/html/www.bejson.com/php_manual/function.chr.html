chr

(PHP 4, PHP 5, PHP 7, PHP 8)

chr — 返回指定的字符

说明

chr(int $ascii): string

返回相对应于 ascii 所指定的单个字符。

此函数与 ord() 是互补的。

参数

ascii

Ascii 码。

返回值

返回规定的字符。

范例

示例 #1 chr() 例子

<?php
$str = "The string ends in escape: ";
$str .= chr(27); /* 在 $str 后边增加换码符 */

/* 通常这样更有用 */

$str = sprintf("The string ends in escape: %c", 27);
?>
