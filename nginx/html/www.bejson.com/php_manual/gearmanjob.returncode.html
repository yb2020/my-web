GearmanJob::returnCode

(PECL gearman >= 0.5.0)

GearmanJob::returnCode — Get last return code

说明

public GearmanJob::returnCode(): int

Returns the last return code issued by the job server.

参数

此函数没有参数。

返回值

A valid Gearman return code.
