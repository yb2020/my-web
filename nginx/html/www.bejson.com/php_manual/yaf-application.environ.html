Yaf_Application::environ

(Yaf >=1.0.0)

Yaf_Application::environ — 获取当前Yaf_Application的环境名

说明

public Yaf_Application::environ(): void

返回当前Yaf_Application的环境名，它被定义在yaf.environ，默认值为"product"。

参数

此函数没有参数。

返回值

范例

示例 #1 Yaf_Application::environ()example

<?php
$config = array(
    "application" => array(
        "directory" => realpath(dirname(__FILE__)) . "/application",
    ),
);

/** Yaf_Application */
$application = new Yaf_Application($config);
print_r($application->environ());
?>

以上例程的输出类似于：

product
