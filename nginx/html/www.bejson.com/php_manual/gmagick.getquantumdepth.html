Gmagick::getquantumdepth — Returns the Gmagick quantum depth as a string

说明

public Gmagick::getquantumdepth(): array

Returns the Gmagick quantum depth as a string.

参数

此函数没有参数。

Returns the Gmagick quantum depth as a string.
