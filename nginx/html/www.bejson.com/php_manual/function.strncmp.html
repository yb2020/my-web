strncmp

(PHP 4, PHP 5, PHP 7, PHP 8)

strncmp — 二进制安全比较字符串开头的若干个字符

说明

strncmp(string $str1, string $str2, int $len): int

该函数与 strcmp() 类似，不同之处在于你可以指定两个字符串比较时使用的长度（即最大比较长度）。

注意该比较区分大小写。

参数

str1

第一个字符串。

str2

第二个字符串。

len

最大比较长度。

返回值

如果 str1 小于 str2 返回 < 0； 如果 str1 大于 str2 返回 > 0；如果两者相等，返回 0。
