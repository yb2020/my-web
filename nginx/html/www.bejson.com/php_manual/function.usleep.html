usleep

(PHP 4, PHP 5, PHP 7, PHP 8)

usleep — 以指定的微秒数延迟执行

说明

usleep(int $micro_seconds): void

以指定的微秒数延缓程序的执行。

参数

micro_seconds

暂停的时间以微秒计。1微秒（micro second）是百万分之一秒。

返回值

没有返回值。

范例

示例 #1 usleep()例子

<?php

// Current time
echo date('h:i:s') . "\n";

// wait for 2 seconds
usleep(2000000);

// back!
echo date('h:i:s') . "\n";

?>

以上例程会输出：

11:13:28
11:13:30

参见

  • sleep() - 延缓执行
  • time_nanosleep() - 延缓执行若干秒和纳秒
  • time_sleep_until() - 使脚本睡眠到指定的时间为止。
  • set_time_limit() - 设置脚本最大执行时间
