sodium_memzero

(PHP 7 >= 7.2.0, PHP 8)

sodium_memzero — Overwrite a string with NUL characters

说明

sodium_memzero(string &$string): void

sodium_memzero() zeroes out the string that is passed by reference.

参数

string

String.

返回值

没有返回值。
