(PECL fann >= 1.0.0)

fann_set_cascade_min_cand_epochs — 设置最小候选周期

说明

fann_set_cascade_min_cand_epochs(resource $ann, int $cascade_min_cand_epochs): bool

设置最小候选周期。

参数

ann

神经网络 资源。

cascade_min_cand_epochs

设置最小候选周期。
