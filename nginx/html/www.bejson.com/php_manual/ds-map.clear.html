Ds\Map::clear

(PECL ds >= 1.0.0)

Ds\Map::clear — Removes all values

说明

public Ds\Map::clear(): void

Removes all values from the map.

参数

此函数没有参数。

返回值

没有返回值。

范例

示例 #1 Ds\Map::clear() example

<?php
$map = new \Ds\Map([
    "a" => 1,
    "b" => 2,
    "c" => 3,
]);
print_r($map);

$map->clear();
print_r($map);
?>
