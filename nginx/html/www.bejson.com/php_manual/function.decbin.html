decbin

(PHP 4, PHP 5, PHP 7, PHP 8)

decbin — 十进制转换为二进制

说明

decbin(int $number): string

返回一字符串，包含有给定 number 参数的二进制表示。所能转换的最大数值为十进制的 4294967295，其结果为 32 个 1 的字符串。

参数

number

Decimal value to convert

返回值

范例

示例 #1 decbin() 例子

<?php
echo decbin(12) . "\n";
echo decbin(26);
?>

以上例程会输出：

1100
11010
