bin2hex

(PHP 4, PHP 5, PHP 7, PHP 8)

bin2hex — 函数把包含数据的二进制字符串转换为十六进制值

说明

bin2hex(string $str): string

把二进制的参数 str 转换为的十六进制的字符串。转换使用字节方式，高四位字节优先。

参数

str

二进制字符串。

返回值

返回指定字符串十六进制的表示。

参见

  • hex2bin() - 转换十六进制字符串为二进制字符串
  • pack() - 将数据打包成二进制字符串
