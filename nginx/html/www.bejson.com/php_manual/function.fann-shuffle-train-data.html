fann_shuffle_train_data

(PECL fann >= 1.0.0)

fann_shuffle_train_data — 打算训练数据，使顺序随机。

说明

fann_shuffle_train_data(resource $train_data): bool

打算训练数据，使顺序随机。 增量训练时推荐使用，但是批训练时并没有什么明显的效果。

参数

train_data

神经网络训练数据 资源。

返回值

成功时返回 true，其它情况下返回 false。
