cos

(PHP 4, PHP 5, PHP 7, PHP 8)

cos — 余弦

说明

cos(float $arg): float

cos() 返回参数 arg 的余弦值。参数 arg 的单位为弧度。

参数

arg

以弧度表示的角度

返回值

arg的余弦值

范例

示例 #1 cos() 例子

<?php

echo cos(M_PI); // -1

?>

参见

  • acos() - 反余弦
  • sin() - 正弦
  • tan() - 正切
  • deg2rad() - 将角度转换为弧度
