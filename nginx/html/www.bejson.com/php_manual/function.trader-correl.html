trader_correl

(PECL trader >= 0.2.0)

trader_correl — Pearson's Correlation Coefficient (r)

说明

trader_correl(array $real0, array $real1, int $timePeriod = ?): array

参数

real0

浮点数数组。

real1

浮点数数组。

timePeriod

Number of period. Valid range from 2 to 100000.

返回值

Returns an array with calculated data or false on failure.
