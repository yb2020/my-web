xml_error_string

(PHP 4, PHP 5, PHP 7, PHP 8)

xml_error_string — 获取 XML 解析器的错误字符串

说明

xml_error_string(int $code): string

根据给定的 code 获得 XML 解析器错误字符串。

参数

code

由 xml_get_error_code() 返回的错误代码。

返回值

返回与 code 描述的错误代码参数对应的文本描述字符串，若没有与之对应的描述，则返回 false。
