ldap_count_references

(PHP 8)

ldap_count_references — Counts the number of references in a search result

说明

ldap_count_references(LDAP\Connection $ldap, LDAP\Result $result): int

Counts the number of references in a search result.

参数

ldap

通过 ldap_connect() 返回的 LDAP\Connection 实例。

result

通过 ldap_list() 或者 ldap_search() 返回的 LDAP\Result 实例。

返回值

Returns the number of references in a search result.

更新日志

版本 说明
8.1.0 现在 ldap 参数接受 LDAP\Connection 实例，之前接受 资源(resource)。
8.1.0 The result parameter expects an LDAP\Result instance now; previously, a 资源(resource) was expected.
