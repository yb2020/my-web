dgettext

(PHP 4, PHP 5, PHP 7, PHP 8)

dgettext — Override the current domain

说明

dgettext(string $domain, string $message): string

The dgettext() function allows you to override the current domain for a single message lookup.

参数

domain

The domain

message

The message

返回值

A string on success.
