getcwd

(PHP 4, PHP 5, PHP 7, PHP 8)

getcwd — 取得当前工作目录

说明

getcwd(): string|false

取得当前工作目录。

返回值

成功则返回当前工作目录，失败返回 false。

在某些 Unix 的变种下，如果任何父目录没有设定可读或搜索模式，即使当前目录设定了，getcwd() 还是会返回 false。有关模式与权限的更多信息见 chmod()。

警告

如果在编译 PHP 时启用了 ZTS（Zend Thread Safety）选项，则 getcwd() 函数返回的当前工作目录可能与操作系统接口返回的目录不同。依赖于当前工作目录的外部库（通过 FFI 调用）将受到影响。

范例

示例 #1 getcwd() 例子

<?php

// current directory
echo getcwd() . "\n";

chdir('cvs');

// current directory
echo getcwd() . "\n";

?>

以上例程的输出类似于：

/home/didou
/home/didou/cvs
