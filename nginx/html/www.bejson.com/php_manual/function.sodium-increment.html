sodium_increment

(PHP 7 >= 7.2.0, PHP 8)

sodium_increment — Increment large number

说明

sodium_increment(string &$string): void

Treat the string as a little-endian unsigned integer, then increase it by 1. Constant-time.

参数

string

String to increment.

返回值

没有返回值。
