DOMNode::cloneNode

(PHP 5, PHP 7, PHP 8)

DOMNode::cloneNode — Clones a node

说明

public DOMNode::cloneNode(bool $deep = false): DOMNode|false

Creates a copy of the node.

参数

deep

Indicates whether to copy all descendant nodes. This parameter is defaulted to false.

返回值

The cloned node.
