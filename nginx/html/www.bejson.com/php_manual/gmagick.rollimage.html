Gmagick::rollimage

(PECL gmagick >= Unknown)

Gmagick::rollimage — Offsets an image

说明

public Gmagick::rollimage(int $x, int $y): Gmagick

Offsets an image as defined by x and y.

参数

x

The x offset.

y

The y offset.

返回值

The Gmagick object on success

错误／异常

错误时抛出 GmagickException。
