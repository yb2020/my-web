tanh

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

tanh — 双曲正切

说明

tanh(float $arg): float

返回 arg 的双曲正切值，定义为 sinh(arg)/cosh(arg)。

参数

arg

要处理的参数

返回值

arg 的双曲正切值

参见

  • tan() - 正切
  • atanh() - 反双曲正切
  • sinh() - 双曲正弦
  • cosh() - 双曲余弦
