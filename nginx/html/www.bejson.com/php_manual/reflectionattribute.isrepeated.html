ReflectionAttribute::isRepeated

(PHP 8)

ReflectionAttribute::isRepeated — 返回该名称的属性是否可以重复出现在一个代码元素上

说明

public ReflectionAttribute::isRepeated(): bool

返回该名称的属性是否可以重复出现在一个代码元素上。

参数

此函数没有参数。

返回值

当属性被重复使用时返回 true，否则返回 false。
