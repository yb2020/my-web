Swoole\Process::kill

(PECL swoole >= 1.9.0)

Swoole\Process::kill — Send signal to the child process.

说明

public static Swoole\Process::kill(int $pid, int $signal_no = ?): bool

Send signal to the child process.

参数

pid

Process pid

signal_no

Signal to be sent

返回值

成功时返回 true， 或者在失败时返回 false。
