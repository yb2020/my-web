FANNConnection::__construct

(PECL fann >= 1.0.0)

FANNConnection::__construct — 连接构造器

说明

public FANNConnection::__construct(int $from_neuron, int $to_neuron, float $weight)

创建一个新的连接并且初始化它的参数。一旦连接被初始化，只有其权重能被修改。

参数

from_neuron

开始连接的神经元。

to_neuron

结束连接的神经元。

weight

连接的权重。
