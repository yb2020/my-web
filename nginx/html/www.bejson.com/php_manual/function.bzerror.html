bzerror

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

bzerror — 返回包含 bzip2 错误号和错误字符串的一个 array

说明

bzerror(resource $bz): array

返回文件指针中返回的 bzip2 错误的错误号和错误字符串。

参数

bz

文件指针。它必须是有效的并且指向 bzopen() 成功打开的文件。

返回值

返回一个关联数组，包含错误码于 errno 条目， 以及错误信息于 errstr 条目。

范例

示例 #1 bzerror() 范例

<?php
$error = bzerror($bz);

echo $error["errno"];
echo $error["errstr"];
?>
