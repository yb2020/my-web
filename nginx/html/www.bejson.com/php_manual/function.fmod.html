fmod

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

fmod — 返回除法的浮点数余数

说明

fmod(float $x, float $y): float

返回被除数（x）除以除数（y）所得的浮点数余数。余数（r）的定义是：x = i * y + r，其中 i 是整数。如果 y 是非零值，则 r 和 x 的符号相同并且其数量值小于 y。

参数

x

被除数

y

除数

返回值

x/y的浮点数余数。

范例

示例 #1 fmod() 的使用

<?php
$x = 5.7;
$y = 1.3;
$r = fmod($x, $y);
// $r equals 0.5, because 4 * 1.3 + 0.5 = 5.7
?>
