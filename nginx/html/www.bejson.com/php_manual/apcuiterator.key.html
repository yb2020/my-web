APCUIterator::key

(PECL apcu >= 5.0.0)

APCUIterator::key — Get iterator key

说明

public APCUIterator::key(): string

Gets the current iterator key.

参数

此函数没有参数。

返回值

Returns the key on success, or false upon failure.
