Ds\Stack::clear

(PECL ds >= 1.0.0)

Ds\Stack::clear — Removes all values

说明

public Ds\Stack::clear(): void

Removes all values from the stack.

参数

此函数没有参数。

返回值

没有返回值。

范例

示例 #1 Ds\Stack::clear() example

<?php
$stack = new \Ds\Stack([1, 2, 3]);
print_r($stack);

$stack->clear();
print_r($stack);
?>

以上例程的输出类似于：

Ds\Stack Object
(
    [0] => 3
    [1] => 2
    [2] => 1
)
Ds\Stack Object
(
