tan

(PHP 4, PHP 5, PHP 7, PHP 8)

tan — 正切

说明

tan(float $arg): float

tan() 返回参数 arg 的正切值。参数 arg 的单位为弧度。

参数

arg

要处理的以弧度为单位的参数

返回值

arg 的正切值

范例

示例 #1 tan() 例子

<?php

echo tan(M_PI_4); // 1

?>

参见

  • atan() - 反正切
  • atan2() - 两个参数的反正切
  • sin() - 正弦
  • cos() - 余弦
  • tanh() - 双曲正切
  • deg2rad() - 将角度转换为弧度
