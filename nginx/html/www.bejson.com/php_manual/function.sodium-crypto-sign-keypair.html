sodium_crypto_sign_keypair

(PHP 7 >= 7.2.0, PHP 8)

sodium_crypto_sign_keypair — Randomly generate a secret key and a corresponding public key

说明

sodium_crypto_sign_keypair(): string

Generate a random Ed25519 keypair as one string.

参数

此函数没有参数。

返回值

Ed25519 keypair.
