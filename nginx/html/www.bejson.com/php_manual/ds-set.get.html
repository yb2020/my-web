Ds\Set::get

(PECL ds >= 1.0.0)

Ds\Set::get — Returns the value at a given index

说明

public Ds\Set::get(int $index): mixed

Returns the value at a given index.

参数

index

The index to access, starting at 0.

返回值

The value at the requested index.

错误／异常

OutOfRangeException if the index is not valid.

范例

示例 #1 Ds\Set::get() example

<?php
$set = new \Ds\Set(["a", "b", "c"]);

var_dump($set->get(0));
var_dump($set->get(1));
var_dump($set->get(2));
?>

以上例程的输出类似于：

string(1) "a"
string(1) "b"
string(1) "c"

示例 #2 Ds\Set::get() example using array syntax

<?php
$set = new \Ds\Set(["a", "b", "c"]);

var_dump($set[0]);
var_dump($set[1]);
var_dump($set[2]);
?>

以上例程的输出类似于：

string(1) "a"
string(1) "b"
string(1) "c"
