sodium_pad

(PHP 7 >= 7.2.0, PHP 8)

sodium_pad — Add padding data

说明

sodium_pad(string $string, int $block_size): string

Right-pad a string to a desired length. Timing-safe.

参数

string

Unpadded string.

block_size

The string will be padded until it is an even multiple of the block size.

返回值

Padded string.
