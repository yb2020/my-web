convert_uudecode

(PHP 5, PHP 7, PHP 8)

convert_uudecode — 解码一个 uuencode 编码的字符串

说明

convert_uudecode(string $data): string

convert_uudecode() 解码一个 uuencode 编码的字符串。

参数

data

uuencode 编码后的数据

返回值

返回解码后的字符串数据， 或者在失败时返回 false.。

范例

示例 #1 convert_uudecode() 例子

<?php
/* 你猜会输出啥？:) */
echo convert_uudecode("+22!L;W9E(%!(4\"$`\n`");
?>
