zip_entry_filesize

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.0.0)

zip_entry_filesize — 检索目录实体的实际大小

说明

zip_entry_filesize(resource $zip_entry): int

返回指定目录实体的实际大小。

参数

zip_entry

由函数zip_read() 返回的目录实体。

返回值

返回该目录实体的大小。

参见

  • zip_open() - 打开ZIP存档文件
  • zip_read() - 读取ZIP存档文件中下一项
