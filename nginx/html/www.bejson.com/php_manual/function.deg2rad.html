deg2rad

(PHP 4, PHP 5, PHP 7, PHP 8)

deg2rad — 将角度转换为弧度

说明

deg2rad(float $number): float

本函数把 number 从角度转换成弧度。

参数

number

以角度为单位的值

返回值

number 等量的弧度值

范例

示例 #1 deg2rad() 例子

<?php

echo deg2rad(45); // 0.785398163397
var_dump(deg2rad(45) === M_PI_4); // bool(true)

?>
