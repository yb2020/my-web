ReflectionEnumBackedCase::getBackingValue

(PHP 8 >= 8.1.0)

ReflectionEnumBackedCase::getBackingValue — 获取枚举条目回退的标量值

说明

public ReflectionEnumBackedCase::getBackingValue(): string|int

获取枚举条目回退的标量值。

参数

此函数没有参数。

返回值

枚举条目对应的标量值。

范例

示例 #1 ReflectionEnum::getBackingValue() 例子

<?php
enum Suit: string
{
    case Hearts = 'H';
    case Diamonds = 'D';
    case Clubs = 'C';
    case Spades = 'S';
}

$rEnum = new ReflectionEnum(Suit::class);

$rCase = $rEnum->getCase('Spades');

var_dump($rCase->getBackingValue());
?>

以上例程会输出：

string(1) "S"
