imagefilledrectangle

(PHP 4, PHP 5, PHP 7, PHP 8)

imagefilledrectangle — 画一矩形并填充

说明

imagefilledrectangle(
    resource $image,
    int $x1,
    int $y1,
    int $x2,
    int $y2,
    int $color
): bool

imagefilledrectangle() 在 image 图像中画一个用 color 颜色填充了的矩形，其左上角坐标为 x1，y1，右下角坐标为 x2，y2。0, 0 是图像的最左上角。
