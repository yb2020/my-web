filter_id

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

filter_id — 返回与某个特定名称的过滤器相关联的id

说明

filter_id(string $filtername): int

参数

filtername

待获取的过滤器名称

返回值

如果获取成功则返回过滤器id，如果过滤器不存在则返回 false 。

  • filter_list() - 返回所支持的过滤器列表
