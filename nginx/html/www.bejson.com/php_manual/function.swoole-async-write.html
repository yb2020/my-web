swoole_async_write

(PECL swoole >= 1.9.0)

swoole_async_write — Write data to a file stream asynchronously

说明

swoole_async_write(
    string $filename,
    string $content,
    int $offset = ?,
    callable $callback = ?
): bool

参数

filename

The filename being written.

content

The content writing to the file.

offset

The offset.

callback

返回值

成功时返回 true， 或者在失败时返回 false。
