fann_get_quickprop_decay

(PECL fann >= 1.0.0)

fann_get_quickprop_decay — 返回衰退值，用于在 quickprop 训练迭代时衰减权重

说明

fann_get_quickprop_decay(resource $ann): float

衰退值是一个小的负数，用于在 quickprop 训练迭代时衰减权重。 这是用来确保训练时权重不要太高。

默认衰退值是 -0.0001.

参数

ann

神经网络 资源。

返回值

成功，返回衰退值，错误则返回 false .
