关键词列表

这些词语在 PHP 中有着特殊的意义。它们中有些像是函数，有些像是常量，但是它们不是真的，它们只是语言结构的一部分。不能使用它们的任何一个作为常量、类名、函数或方法名。但是它们可以被用作类的属性、常量以及类的方法名，或者接口名和 traits 名，除了 class 不能被用作常量名。
