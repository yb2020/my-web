Imagick::clampImage

(PECL imagick 3 >= 3.3.0)

Imagick::clampImage — Description

说明

public Imagick::clampImage(int $channel = Imagick::CHANNEL_DEFAULT): bool

Restricts the color range from 0 to the quantum depth.

参数

channel

返回值

成功时返回 true， 或者在失败时返回 false。
