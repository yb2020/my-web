ldap_mod_add_ext

(PHP 7 >= 7.3.0, PHP 8)

ldap_mod_add_ext — Add attribute values to current attributes

说明

ldap_mod_add_ext(
    LDAP\Connection $ldap,
    string $dn,
    array $entry,
    ?array $controls = null
): LDAP\Result|false

Does the same thing as ldap_mod_add() but returns an LDAP\Result instance to be parsed with ldap_parse_result().

参数

See ldap_mod_add()

返回值

Returns an LDAP\Result instance, 或者在失败时返回 false.
