WeakMap::offsetSet

(PHP 8)

WeakMap::offsetSet — 更新 map 新的键值对

说明

public WeakMap::offsetSet(object $object, mixed $value): void

更新 map 新的键值对。如果 key 已经在 map 中存在， 新的值会替换老的值。

参数

object

键值对中作为 key 的对象。

value

任意数据作为键值对的值。

返回值

没有返回值。
