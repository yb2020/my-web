互联网协议和支持¶

本章介绍的模块实现了互联网协议以及相关技术支持。 它们都是用 Python 实现的。 这些模块大多需要依赖于系统的模块 socket 作为前提，该模块在大多数流行系统平台上都受到支持。 下面是一份概览:

    • wsgiref.types -- WSGI types for static type checking
